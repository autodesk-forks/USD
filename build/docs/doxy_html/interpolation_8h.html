◆ USD_LINEAR_INTERPOLATION_TYPES

#define USD_LINEAR_INTERPOLATION_TYPES

Sequence of value types that support linear interpolation. These types and VtArrays of these types are supported:

  • GfHalf
  • float
  • double
  • SdfTimeCode
  • GfMatrix2d
  • GfMatrix3d
  • GfMatrix4d
  • GfVec2d
  • GfVec2f
  • GfVec2h
  • GfVec3d
  • GfVec3f
  • GfVec3h
  • GfVec4d
  • GfVec4f
  • GfVec4h
  • GfQuatd (via quaternion slerp)
  • GfQuatf (via quaternion slerp)
  • GfQuath (via quaternion slerp)
