Compute joint indices corresponding to each point in a bone mesh.

This can be used to animate a bone mesh using normal skinning algos. This does not compute joint weights (they would all be 1s). The numPoints corresponds to the number of points computed by UsdSkelImagingComputeBoneTopology.
