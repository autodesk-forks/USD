Detailed Description

The base class for all typed schemas (those that can impart a typeName to a UsdPrim), and therefore the base class for all concrete, instantiable "IsA" schemas.

UsdTyped implements a typeName-based query for its override of UsdSchemaBase::_IsCompatible(). It provides no other behavior.
