Detailed Description

Common structure used to report registered plugins in one of the plugin registries.

The id token is used for internal api communication about the name of the plugin. displayName is a human readable name given to the plugin intended to be used in menus. priority is used to provide an ordering of plugins. The plugin with the highest priority is determined to be the default (unless overridden by the application). In the event of a tie the string version of id is used to sort alphabetically ('a' has priority over 'b').
