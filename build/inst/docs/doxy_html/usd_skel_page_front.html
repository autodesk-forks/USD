UsdSkel : USD Skeleton Schema and API

UsdSkel defines schemas and API that form a basis for interchanging both skeletally skinned meshes and joint animations between DCC tools in a graphics pipeline.

API Manual

This manual contains API documentation for the UsdSkel module, along with an introduction key schemas and API concepts.
