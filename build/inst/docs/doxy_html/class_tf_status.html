Detailed Description

Represents an object that contains information about a status message.

See Guide To Diagnostic Facilities in the C++ API reference for a detailed description of the status message issuing API. For a example of how to post a status message, see TF_STATUS(), also in the C++ API reference.

In the Python API, you can issue a status message with Tf.Status().
