Performs registrations required for the specified package resolver class to be discovered by Ar's plugin mechanism.

This typically would be invoked in the source file defining the resolver class. For example:

// in .cpp file
AR_DEFINE_PACKAGE_RESOLVER(CustomPackageResolverClass, ArPackageResolver);
