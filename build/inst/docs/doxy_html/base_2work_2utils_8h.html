◆ WorkSwapDestroyAsync()

void WorkSwapDestroyAsync ( T &  obj)

Swap obj with a default-constructed T instance, return and arrange for the swapped-out instance to be destroyed asynchronously.

This means that any code that obj's destructor might invoke must be safe to run both concurrently with other code and at any point in the future. This might not be true, for example, if obj's destructor might try to update some other data structure that could be destroyed by the time obj's destruction occurs. Be careful.
