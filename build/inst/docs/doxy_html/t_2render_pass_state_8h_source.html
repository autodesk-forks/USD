204  // A 4d-vector v encodes a 2d-transform as follows:
205  // (x, y) |-> (v[0] * x + v[2], v[1] * y + v[3]).
206  using AxisAlignedAffineTransform = GfVec4f;

208  // Computes the transform from pixel coordinates to the horizontally
209  // normalized filmback space which has the following properties:
210  // 1. x = -1 and +1 corresponds to the left and right edge of the filmback,
211  // respectively.
212  // 2. (0, 0) corresponds to the center of the filmback.
213  // 3. Moving a unit in either the x- or y-direction moves by the same
214  // distance on the filmback. In other words, y = -1/a and +1/a
215  // corresponds to the bottom and top edge of the filmback, respectively,
216  // where a is the camera's aspect ratio.
