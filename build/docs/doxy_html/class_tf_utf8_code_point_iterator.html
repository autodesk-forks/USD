Detailed Description

Defines an iterator over a UTF-8 encoded string that extracts unicode code point values.

UTF-8 is a variable length encoding, meaning that one Unicode character can be encoded in UTF-8 as 1, 2, 3, or 4 bytes. This iterator takes care of iterating the necessary characters in a string and extracing the Unicode code point of each UTF-8 encoded character in the sequence.
